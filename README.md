# IPFURB20261
Introdução Programação - Matutino

Repositório contendo exercício da matéria de Introdução à Programação. 

O Exercício 7 da Unidade 3 diz respeito ao desenvolvimento de um código para realizar a contagem de litros de refrigerante comprados por um determinado comerciante conforme os três formatos de venda do produto pela fábrica.
